The data for this project was taken from GSMArena, the Mobile Phone Museum, Phone Scoop, and sometimes the Legacy Portable Computing Wiki. Data from individual sites was often incomplete or inconsistent, so sites were usually thoroughly cross-referenced for each individual entry.

Photos used are either from Wikimedia Commons, the Mobile Phone Museum, or original sales photos from phone brands. Minimal editing for image consistency was done with Canva and remove.bg.

The design of this site was inspired by the website of artist Yoshitomo Nara. Custom scrollbar code is from W3Schools.

This project was made by Maija Chover. Take a look at the GitHub page here.
